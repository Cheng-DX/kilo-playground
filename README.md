# iOS 26 液态玻璃效果演示

## 项目概述

这个项目是一个交互式的HTML/CSS/JavaScript演示，旨在展示和解释iOS 26中引入的"液态玻璃"（Liquid Glass）视觉效果特性。

## 什么是液态玻璃？

液态玻璃是iOS 26的一项突破性视觉技术，它结合了以下核心特性：

### 主要特征

1. **玻璃形态效果 (Glass Morphism)**
   - 半透明背景
   - 背景模糊处理
   - 细微的边框高光
   - 深度感和层次感

2. **液态动画效果**
   - 流体般的过渡动画
   - 涟漪效果
   - 波浪运动
   - 形态变换

3. **交互响应**
   - 鼠标/触摸反馈
   - 实时变形
   - 动态透明度变化
   - 视差效果

## 技术实现

### CSS 技术
- `backdrop-filter`: 实现背景模糊效果
- `linear-gradient`: 创建玻璃质感
- `animation` 和 `@keyframes`: 实现流畅动画
- `transform`: 3D变换和透视效果
- `filter`: 模糊和色彩调整

### JavaScript 功能
- 交互式涟漪生成
- 鼠标跟随效果
- 幻灯片导航系统
- 动态动画控制
- 粒子效果系统

### 响应式设计
- 移动设备适配
- 触摸事件支持
- 自适应布局
- 性能优化

## 性能优化建议

1. **移动设备**: 减少同时运行的动画数量
2. **低性能设备**: 禁用粒子效果
3. **电池优化**: 暂停后台标签页的动画
